1 ) DNS 조회 - 너디너리 홈페이지 도메인을 IP 주소로 변경

2 ) TCP 연결 설정 (Three-way Handshake) - 클라이언트가 서버의 443 포트에 SYN 패킷을 보냄. 

     →서버는 SYN + ACK 응답 → 클라이언트가 ACK 응답 → 연결 설정 완료

3 ) HTTP 요청 전송 - TLS/TCP 위에서 클라이언트가 서버에 HTTP 요청

4 ) 서버 도착 및 처리- 서버(너디너리 웹 서버)가 해당 IP와 포트(443)에서 요청을 수신.

      -TCP 레벨에서 데이터 재조립 → 애플리케이션 계층(HTTP 요청)으로 전달.

1. 응답 및 페이지 표시 
    
    -서버는 준비된 HTTP 응답 메시지를 TCP를 통해 클라이언트(브라우저)에 전송.
    
    -클라이언트 브라우저는 TCP 계층에서 데이터를 받아 복원 → HTTP 응답 해석.
