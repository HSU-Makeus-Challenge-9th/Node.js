1. 너디너리 홈페이지 접속하는 과정 적어보기 (소켓프로그래밍과 같은 개념 없이 TCP, IP, PORT 등의 개념 등 오늘 배운 내용으로 간단하게 적어주세요)



1. 클라이언트(사용자)가 브라우저에 너디너리 홈페이지 검색
2. 브라우저가 DNS 서버에 너디너리 홈페이지의 IP주소를 알아냄(100.111.100)
3. 클라이언트는 계층구조(TCP/IP 5계층)를 거쳐 데이터를 생성(패킷)
    1. 패킷에는 클라이언트의 ip주소와 너디너리 홈페이지의 ip주소, port 번호 등을 가지고 있음
4. Access Network(와이파이, 랜선 등)를 통해 Network Core(라우터, 스위치 등)로 전송
    1. 라우터는 패킷을 분석 해 목적지 ip가 서버임을 확인하고 최적 경로를 통해 패킷 단위로 이동
5. 서버 도착
    1. 서버에 도착하면 패킷을 분석해 TCP 연결(3way handshake)을 수행
    2. 목적지의 포트번호고 443이므로 https 프로세스 수행
6. 서버는 요청한 HTML/CSS 파일을 클라이언트에게 다시 전송
    1. 이때 역시 AccessNetwork→Network Core → Access Network → 클리이언트로 역방향 이동
7. 클라이언트 브라우저는 받은 데이터를 렌더링
